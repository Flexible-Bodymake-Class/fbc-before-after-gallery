# Before / After Gallery（モック版）

Flexible Bodymake Class 会員のみなさんの「変化の記録」を讃えるためのギャラリーページです。
現在は最終デザイン確認前の下書き（モック）です。

公開URL: https://flexible-bodymake-class.github.io/fbc-before-after-gallery/

## 今のバージョンについて

- 掲載されている写真・お名前・コメントはすべてサンプル（ダミー）です。実際の会員の方の情報は一切含まれていません。
- 検索エンジンには表示されない設定にしています。このURLを知っている方だけが見られる状態です。

## 見ていただきたいポイント

- 全体の雰囲気（色づかい・余白・言葉づかい）がFBCらしいか
- 変化を讃える「新しく仲間入りした変化」の見せ方
- テーマタグ（土台編・スワン・ブリッジなど）の項目や並びに過不足がないか
- 一番下のフォームへの案内文・Discordへの案内文の言葉づかい

## 今後の流れ（予定）

1. このデザインについてタイチさんからフィードバックをいただく
2. Googleフォームを用意し、会員の方がBefore/After写真とコメントを投稿できるようにする
3. タイチさんが投稿を確認し、ひと言メッセージを添えて承認する
4. 承認された投稿が自動的にこのページに反映される（写真・コメントの手動アップロードは不要）
5. 反映と同時に、Discordのビフォーアフター部屋にもお祝い通知が届く

ご不明な点があれば、いつでも姫までご連絡ください。
